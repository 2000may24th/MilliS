# MilliS: 멀티태스커를 위한 서버 타이머 앱

MilliS는 서버와 동기화된 현재 시각을 밀리초(ms) 단위까지 정확하게 확인할 수 있는 iOS 애플리케이션입니다.

서버 기준 시각을 기반으로 시간을 계산하여 제공하며, Live Activities를 통해 앱을 실행하지 않은 상태에서도 Dynamic Island와 잠금 화면에서 실시간 서버 시간을 확인할 수 있습니다.


## MilliS가 필요한 이유

데스크탑에서는 서버 시간 웹사이트를 별도의 창이나 모니터에 띄워두고 다른 작업을 동시에 수행하는 것이 어렵지 않습니다. 그러나 스마트폰에서는 하나의 앱만 전면에서 사용할 수 있기 때문에, 서버 시간을 확인하려면 현재 사용 중인 브라우저나 앱을 벗어나야 합니다.

티켓 예매, 선착순 구매, 이벤트 참여처럼 찰나의 순간이 중요한 상황​에서는 전환 자체가 불편합니다.

MilliS는 이러한 모바일 환경의 제약을 해결하기 위해 만들어졌습니다.

서버와 주기적으로 시간을 동기화한 뒤, Live Activities와 Dynamic Island를 통해 서버 시각을 잠금 화면과 화면 상단에 지속적으로 표시합니다. 덕분에 사용자는 다른 앱을 이용하는 동안에도 기준 시간을 실시간으로 확인할 수 있으며, 필요한 순간에 앱을 다시 열지 않고도 정확한 서버 시간을 확인할 수 있습니다.


## 주요 기능

### ⏱️ 밀리초 단위 서버 시간 표시

서버 기준 시각을 동기화하여 밀리초(1/1000초) 단위까지 정확한 현재 시각을 제공합니다.

### 📱 Live Activities 지원

잠금 화면에서 서버 시간을 실시간으로 확인할 수 있습니다.

### 🏝️ Dynamic Island 지원

지원 기기에서는 Dynamic Island를 통해 앱을 열지 않고도 현재 서버 시간을 확인할 수 있습니다.

### 🔄 자동 시간 동기화 및 보정

주기적으로 서버와 시간을 동기화하여 사용자가 수동으로 설정하지 않더라도 정확도를 유지합니다.

### ⚡ 가볍고 빠른 동작

최소한의 네트워크 사용으로 정확한 시간을 제공하며 WatchOS를 지원합니다.

### 🌙 다크 모드 및 Liquid Glass 지원

최신 iOS 시스템 설정에 맞추어 화면 모드 변경을 지원합니다.


## 활용 사례

* 콘서트 및 스포츠 경기 티켓 예매
* 대학 수강신청
* 선착순 이벤트 참여
* 한정 수량 상품 구매
* 서버 시간 확인이 필요한 온라인 서비스
* 각종 카운트다운 및 시간 관리


### 지원 기능

* 잠금 화면에서 실시간 서버 시간 확인
* Dynamic Island 표시
* 앱을 종료한 상태에서도 시간 확인 가능
* 빠른 접근성과 높은 가독성
* Apple Watch 동기화(유료)
